# 環境変数リスト

設定のための環境変数のリストです。

## Example

以下の例を参考にしてください。

| 名前         | 説明                                             | 指定しなかったときの挙動       |
| ------------ | ------------------------------------------------ | ------------------------------ |
| `GREET_NAME` | あいさつ相手の名前を指定。                       | デフォルトの名前「hoge」を使用 |
| `API_URL`    | あいさつ文を取得する API サーバのアドレスを指定  | panic を発生させ、異常終了     |
